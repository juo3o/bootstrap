02_부트스트랩 글자와 문단관련

부트스트랩에서는 기본 글자크기가 16px. 행높이(줄간격) 1.428이다

위 내용은 문단태그(p)요소에 자동으로 적용되며, 하단여백(margin)은 16px이 기본.

h1 - 40px

h2 - 32px

h3 - 28px

h4 - 24px

h5 - 20px
h6 - 16px

제목태그(h)요소는 margin-bottom : 8px 이 기본.

small - 글자를 작게 설정 - 14px
mark - 글자에 배경색(부트스트랩에서는 색상이 연함) 적용 - 16px
abbr - 약어나 줄임말 - 16px

blockquote - 인용문 - 14px / #6c757d
dt - 용어제목 - 16px. 진하게
dd - 용어설명 - 16px
code - 문서안에서 태그를 표시. 색상이 자주색으로 변경 - 14px
let gnb = document.querySelector('.gnb');

p태그에 text-muted - 텍스트의 중요도에 따라 문자색상 : 회색

p태그에 text-primary - 텍스트의 중요도에 따라 문자색상 : 파랑

p태그에 text-success - 텍스트의 중요도에 따라 문자색상 : 초록

p태그에 text-info - 텍스트의 중요도에 따라 문자색상 : 하늘

p태그에 text-warning - 텍스트의 중요도에 따라 문자색상 : 주황

p태그에 text-danger - 텍스트의 중요도에 따라 문자색상 : 빨강

p태그에 bg-info - 백그라운드에 배경색(text대신 bg) : 하늘

문단관련 서식

class=lead : 단락 눈에띄게 - 20px

class=small : 상위문자크기보다 85% 작게 - 14px

class=text-start : 왼쪽정렬 - 16px

class=text-end : 오른쪽정렬 - 16px

class=text-center : 중간정렬 - 16px

class=text-justify : 양쪽배분정렬 - 16px

class=text-nowrap :문단 다음줄로 넘어가지 않게 - 16px

class=text-uppercase : 대문자 출력

class=text-lowercase : 소문자 출력

class=text-capitalize : 영문자 첫글자만 대문자로 출력

    ul class=list-unstyled : 목록기호 제거
