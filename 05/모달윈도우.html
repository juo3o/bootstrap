모달윈도우

Modal Window : 사용자 인터페이스 (ui) 디자인 개념에서, 자식에서 부모로 돌아가기 전에 사용자의 상호동작을 요구하는 창

모달 윈도우는 부모창(어두운배경)과 자식요소가 화면의 정가운데 배치되도록하며, 오른족 상단(이미지배너아래)에 닫기버튼을 추가한다

모달 구조

.modal / .modal-dialog / .modal-content / .modal-header / .modal-body / .modal-footer

모달제목

모달내용

모달배경 클릭했을 때 modal 숨기기 :

close 버튼태그에 data-bs-dismiss="modal" 추가하기

모달배경 클릭했을 때 modal 숨겨지지않고 닫기버튼 클릭할 때 숨기기 :

close 버튼태그에 data-bs-dismiss="modal" 추가 + 전체 modal div에 data-bs-backdrop="static" 추가
