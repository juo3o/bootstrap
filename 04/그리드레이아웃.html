    반응형 레이아웃
  1. 가변단위 (%, em, rem, vh, vw)
  2. viewport (디바이스 화면너비에 맞게 컨텐츠를 100%로 맞춤)
  3. mediaQuery (디바이스별 해상도에 따른 서식 CSS)

부트스트랩에서 그리드 레이아웃 속성을 사용하여 반응형 그리드 레이아웃 사용 가능

부트스트랩에서는 하나의 행에 반드시 12개의 열(컬럼)을 사용해야 함

만약 12개의 컬럼을 모두 사용하지 않으려면, 더 넓은 컬럼을 만들기 위해 컬럼들을 하나의 그룹으로 만들어 사용 가능 함

*컬럼 하나당 가로 60px + 양쪽 10px = 80px

짝수 컬럼이 이상적

  • 모바일 사이즈 : 320 - 컬럼 4개 ( 4x8 = 32 )
  • 모바일 사이즈 : 414(아이폰 XR) - 컬럼 5개 (414/8 = 5.)
  • 모바일 사이즈 : 430(아이폰 pro max) - 컬럼 8개 ( 430/8 = 5개) (30/2 = 15 양쪽)
  • PC 최소사이즈 : 1025 ~ - 컬럼 12개(매직넘버)
  • PC 최적사이즈 : 1200 ~ 1400 - 컬럼 14개 or 16개 *
